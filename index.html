Apa itu Simple Project?

Simple Project merupakan project yang dibuat oleh Muhammad Arif Ibrahim dengan tujuan menambah portofolio sekaligus pengalaman dalam mengoding.

Project ini berfokus pada tampilan yang diberikan, sehingga dapat meningkatkan citra sebuah desain website.


Simple project dibuat dengan HTML dan CSS. Saya membuat website ini murni dari awal dan berniat untuk mengembangkan sampai batas saya mendesain website ini.
